### Set to Array in Swift(Set을 Array로 타입을 변경하는 방법)

```swift
let someSet = Set([1, 2, 3])
let someArray = Array(someSet)
print(someArray) // [1, 2, 3]
```

### Sort Array by length in Swift(배열을 길이로 정렬하는 방법)

```swift
let unsortedArray = ["abc", "a", "bc"]
// 길이가 짧은 순으로 정렬(긴 순으로 하려면 부등호를 반대 방향으로 바꿔준다.)
let sortedArray = unsortedArray.sorted { $0.count < $1.count }
print(sortedArray) // ["a", "bc", "abc"]
```

### Remove specific character in String in Swift(문자열에서 특정 문자를 제거하는 방법)

```swift
let string = "{{12, 3}, {1}, {4, 3, 11}}
let removal = ["{", "}"]
let filtered = string.filter { !removal.contains($0) }
print(filtered) // 12, 3, 1, 4, 3, 11
```

### BFS / DFS

- BFS는 Breadth-first search의 약자로 너비 우선 탐색이다.   
    처음으로 주어진 노드의 이웃 노드를 모두 확인하며, 트리 형태로 볼 때 너비를 우선 확인하고  
    아래 단계로 내려가는 양상을 띈다.  
- DFS는 Depth-first search의 약자로 깊이 우선 탐색이다.  
    BFS와 달리 주어진 노드의 가장 먼저 주어진 이웃 노드를 확인하며, 트리 형태로 볼 때 깊이를 우선 확인하고  
    제일 아래에 도착했을 때, 올라오면서 남은 노드를 확인하는 양상을 띈다.

### Switch 문에서 default에서 동작하지 않으려면 어떻게 해야할까?
