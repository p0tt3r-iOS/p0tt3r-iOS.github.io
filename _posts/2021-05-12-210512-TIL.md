### n진법의 문자열을 10진법의 Int형으로 변환하는 방법

n진법으로 만든 문자열은 Int타입의 init?<S>(S, radix: Int) 이니셜라이저로 다시 10진법으로 초기화 할 수 있다.

- 공식문서: Creates a new integer value from the given string and radix.

    ```swift
    let n = 8
    let binary = String(n, radix: 2) // 1000
    let ternary = String(n, radix: 3) // 22
    let binaryToDecimal = Int(String(binary.reversed()), radix: 2) // 8
    let ternaryToDecimal = Int(String(ternary.reversed()), radix: 3) // 8
    ```
