### Enumerated()

- 공식문서: Returns a sequence of pairs (n, x), where n represents a consecutive integer  
starting at zero and x represents an element of the sequence.  
- 딕셔너리, 인덱스셋 등의 컬렉션 타입에 존재하는 메서드로 n은 0부터 시작하는 연속적인 숫자를 표시하고,  
x는 배열의 요소를 나타낸다.(딕셔너리에서는 n은 인덱스, x는 값)  

### CustomStringConvertible

- 공식문서: A type with a customized textual representation.  
- 커스터마이즈된 텍스트 표시를 포함한 타입이라는 의미로 이 프로토콜을 준수하는 타입의 String 표기법을 설정할 수 있다.  

    ```swift
    struct Point {
        let x: Int, y: Int
    }

    // CustomStringConvertible 없이 출력
    let p = Point(x: 21, y: 30)
    print(p)
    // Prints "Point(x: 21, y: 30)"

    // Extension을 통해 CustomStringConvertible 프로토콜 채택
    extension Point: CustomStringConvertible {
        var description: String {
            return "(\(x), \(y))"
        }
    }

    print(p)
    // Prints "(21, 30)"
    ```
