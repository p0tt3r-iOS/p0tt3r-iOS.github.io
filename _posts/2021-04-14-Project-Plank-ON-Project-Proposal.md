# Plank On 기획안(개인 프로젝트)

> 현재 진행하고 있는 개인 프로젝트 Plank On에 대한 기획안이다.  
    기존에 작성해놓았지만, 늦게나마 조금 손을 봐서 올려 놓으려 한다.

## 아이디어

---

- 타겟: 플랭크를 꾸준하게 하지 못하는 분들, 하루에 간단한 플랭크를 한번씩 하길 원하는 분들  
  
- 사용자의 니즈: 간단하게 할 수 있다, 위젯을 통해 매일 진행현황을 확인 할 수 있다  
  
- 제공가치: 꾸준히 플랭크를 하고 있는 나라도 사용할 수 있다.  
  
- 디자인: 최대한 깔끔하고 색상은 다크/일반 모드를 나누고 기본적으로 제공되는 색으로 구성한다.  
  
- 목표: 앱스토어 다운로드 100회 이상  
  
- 기간: 2주(~ 4. 24.)

## 시나리오

---

- 플랭크를 꾸준하기로 한 10-30대의 남녀가 플랭크 ON을 다운로드한다.  
  
- 자신에게 맞는 난이도를 설정하고 원하는 경우, 특정 시간에 알림을 받는다.  
  
- 위젯을 바탕화면에 표시하여 한 날과 안한 날을 표기한다.  
    (문제는 1일차 이후 안했을 떄 5일차에 복귀하면 5일차 시간부터 시작?)  
    (아니면 무조건 28일 일정으로 잡아놓고 시간은 순서대로?)  
  
- 위젯에 빈칸을 다 채우고 싶어지는 사용자는 꾸준히 플랭크를 하게 된다.  
    ( + 매일 오는 알림에 죄책감에 하루라도 더 하게 된다.)  
      
- 꾸준히 해서 상급 과정도 쉬워지면, 자신의 일정을 커스텀할 수 있게 만든다.  

## 기능 리스트

---

- 기술 스택: MVC, UIKit, SwiftUI, Core Data  
  
- 어플 스크린  
    1. 메인 뷰  
        1) 일정(난이도) 설정으로 이동할 수 있는 세그 버튼(네비게이션 뷰)  
        2) 현재 설정한 일정을 표시하는 뷰(컬렉션 뷰?)와 재생 세그 버튼(플랭크 타이머로 이동 / 네비게이션)    
        깃헙 커밋 표 만드는 오픈소스 참조  
        
    2. 일정 설정 뷰 : 난이도 별 일정 설정 세그 버튼(모달) 4개(초급, 중급, 고급, 커스텀) 추가  
        1) 초,중,고급 설정 뷰  
            - 표로(또는 컬렉션 뷰로) 1~28일차까지 진행하는 플랭크 초 표시  
            - 위랑 뷰를 나누어서 알람설정 칸 제작(체크 뷰 구현, 체크 시 선택한 시간에 알람 발송)  
        2) 커스텀 설정 뷰  
            - 표로 1~28일 까지 표시하고 각 날에 초 설정하도록 구현(기본값은 고급 + 30초?)  
            - 알람설정은 위와 동일  
            - 플랭크 타이머 뷰  
                    - 해당 일에 맞는 초를 가진 타이머 상단 표시  
                    - 하단에 재생버튼  
                    - 준비시간 3초(사운드), 끝나면 사운드  
                
- 위젯(SwiftUI)  
    1. 좌측 상단에 앱 아이콘과 똑같은 이미지 뷰  
    2. 하단에 일정표(완료하면 색 변하는) 구현  
    3. 위젯 누르면 앱 실행  
    
- 데이터  
    1. 일정, 완료여부 데이터는 유저 기기에 저장(Core Data)
