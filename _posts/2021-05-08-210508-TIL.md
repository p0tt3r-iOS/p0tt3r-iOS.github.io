### union(_:)

- 공식문서: Returns a new set with the elements of both this and the given set.  
- 기존 셋과 매개변수로 주어진 배열 또는 셋을 합한 새로운 셋을 반환한다.  

### combination(_ length: Int, _ array: [String]) → Set<String>  

- 위 함수는 프로그래머스 소수 찾기(완전탐색)에서 사용한 함수로 Combination(조합)은 기본 형태가 있지만,   
처음 배우는 코드라 작성한다.

    ```swift
    func combination(_ length: Int, _ array: [String]) -> Set<String> {
        if length == 1 { return Set(array) }
        
        var result = Set<String>()
        for i in 0..<array.count {
            let num = array[i]
            var tempArray = array
            tempArray.remove(at: i)
            
            let afterComb = combination(length - 1, tempArray)
            let afterCombArr = afterComb.compactMap { num + $0 }
            result = result.union(afterComb)
            result = result.union(afterCombArr)
            print(result)
        }
        
        return result
    }

    let arr = ["1", "2", "3"]
    print(combination(arr.count, arr)
    // ["23", "32", "13", "132", "321", "213", "123", 
    // "31", "2", "12", "231", "21", "3", "312", "1"]
    // Set 타입이라 순서가 뒤죽박죽이지만, 1, 2, 3으로 만들 수 있는 모든 조합을 반환한다.
    ```
